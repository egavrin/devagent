# Support

## Usage questions

Open a GitHub issue with enough context to reproduce the usage problem or request.

## Bug reports

Use the bug report template for executor, CLI, provider, or verification issues.

## Experimental vs supported

Supported:

- `devagent execute --request --artifact-dir`
- the current `devagent-hub -> devagent-runner -> devagent execute` path

Experimental:

- features or integrations not covered by the validated DevAgent-only workflow
